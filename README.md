# Backend Project

This is a backend project for managing customers, items, orders, and order items.

## Features

- Create, read, update, and delete (CRUD) operations for customers and items.
- Place orders and manage order items.

## Technologies Used

- Spring Boot
- Java
- JPA/Hibernate
- MySQL
